Skipg.me is a work-in-progress

Hey family, Joey here. I told yous guys I was gonna build a site to memorialize Pop. I have big plans, but they are going to take a while. I threw up this placeholder just to let you know that I'm working on it and it will happen soon. How soon is a question I don't know the answer to yet. I'm not using any templates or site generator services like Wix. This is 100% hand-typed code by me in what very little spare time I have. Instead of paying for a hosting service I just rent a small virtual server for $5 a month and I have to configure it to host this site by myself too which is another layer of complexity, but it keeps it cheap. Oh, and by the way, the finished product will look amazing. This is just a placeholder to let you guys know what's up. I'm not going to spend time making this pretty because the time is better spent making creating my vision.

My plans for each section include:

Home
  • It's a landing page so it'll need to look nice.
  • Have obituary PDF available for download.
  • Have any links to obit, Owens & Brumley, etc.
Slideshow
  • Randomly generate endless loop slideshow using all available photos with cool transitions.
  • Maybe have music? (if so...Proud Mary, Dancing in the Street, I've Got You Babe, and some more)
Upload Photos
  • Allow family to upload directly to site and immediately use photos in slideshow and download page.
  • Have upload password protected and share password with family. This will keep any random web surfer that stumbles across the page from uploading whatever they want.
  • EVENTUALLY add backend to save tags and uploader name to allow for photo sorting and filtering.
View Photos
  • Dynamically show photos in a Pinterest-style masonry grid that keeps loading more photos as you scroll.
  • Have each photo show enlarged when clicked and have options for download and adding tags.
  • Have Download All button at top of page that downloads all photos in a zip file (allows for quick, bulk download).
  • EVENTUALLY have sorting options at the top to allow sorting by tag, added date, uploader, etc.
